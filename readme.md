# BeanStore

**Описание:**  
Интернет-магазин кофе с минималистичным дизайном, тёмно-золотой цветовой гаммой и системой фильтрации.

---

## Структура проекта

```plaintext
bean_store/
├── index.html        
├── styles.css        
├── script.js        
└── images/           
    ├── image1.png
    ├── image2.png
    ├── image3.png
    ├── image4.png
    ├── image5.png
    ├── image6.png
    ├── image7.png
    ├── image8.png
    └── image9.png


⸻

Технологии
 • HTML5
 • CSS3: Flexbox, Grid, backdrop-filter, кастомные стили для range и checkbox
 • JavaScript (нативный)

⸻

Реализованный функционал

Интерфейс
 • Шапка с логотипом и навигацией
 • Hero-блок с призывом к действию
 • Каталог из 9 карточек товаров
 • Блок контактов

Фильтрация (UI)
 • Ползунок цены (диапазон от 500 до 2000 ₽)
 • Чекбоксы Наличие: В наличии / Под заказ
 • Чекбоксы Категория: Для себя / Подарочные наборы / Опт
 • Чекбоксы Обжарка: Светлая / Средняя / Тёмная

Интерактив
 • Ползунок цены с синхронизацией значений
 • Защита от пересечения ползунков
 • Ховер-эффекты на карточках товаров и кнопках

Внешний вид
 • Фон: тёмный (#1c1c1c)
 • Акценты: золотой (#e6c7a3)
 • Шрифт заголовков: Playfair Display
 • Шрифт основного текста: Inter
 • Карточки: полупрозрачный фон, рамка, подъём при наведении
 • Скролл: отключён

⸻

Установка и запуск
 • Открыть index.html в браузере
 • Или использовать npx live-server
