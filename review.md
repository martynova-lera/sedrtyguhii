#  Оценка проекта BeanStore

##  Критерии оценки проекта

Проект интернет-магазина кофе BeanStore можно оценить по следующим критериям:

###  Дизайн
- современный и минималистичный интерфейс  
- использование премиальной цветовой палитры (тёмные оттенки + золотые акценты)  
- визуальная привлекательность  

###  Удобство использования (UX)
- понятная структура страницы  
- простая навигация  
- наличие фильтров (цена, категория, обжарка)  

###  Функциональность
- отображение каталога товаров  
- наличие интерактивных элементов  
- базовая фильтрация  

###  Качество кода
- разделение на HTML, CSS и JavaScript  
- читаемость и структурированность кода  
- отсутствие лишних элементов

##  Сравнение с конкурентами

В качестве конкурентов можно рассмотреть:

- Starbucks  
- Lavazza  
- Tchibo  

###  Преимущества BeanStore
- минималистичный и современный дизайн  
- удобная навигация  
- быстрый доступ к товарам  

###  Отличия от конкурентов
- упрощённая структура сайта  
- ориентация на онлайн-продажи  
- лёгкость использования  

###  Возможные улучшения
- добавление полноценной корзины  
- расширенная фильтрация  
- интеграция системы оплаты
